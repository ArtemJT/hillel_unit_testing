# ДЗ 21. Unit-тестирование


1. Создать программу, которая считывает 2 csv, обрабатывает их и сохраняет в новые csv.

2. Проект должен быть построен с помощью maven

3. Использовать Lombok

4. Написать Unit тесты под все критические узлы (обоснованый выбор за разработчиком).

5. Алгоритм работы:

• Считывается 2 файла

• Выполняется парсинг. Примечание. Пропускаем первую строку, с заголовками, дальше читаем построчно и делим строку по делителю ’;’

• Результат каждой строки мапится в соответствующий объект.

• В результате получаем список однотипных объектов из 2-х файлов

• Обрабатываем этот список. Необходимо разделить список в разные списки для каждого магазина. Примечание, можно сделать маппинг с помощью stream api

• Необходимо выполнить слияния однотипных товаров, просуммировать кол-во, и расчитать среднюю цену товара.

• Результат по каждому магазину магазину сохранить в новый csv, например атб_res.csv
